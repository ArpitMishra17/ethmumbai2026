Cross-Chain Verification
Three steps. Two chains. One cryptographic proof that you own your agent.
Base Sepolia
Register on AgentRegistry
Call registerAgent(name, ensName) on Base. Get an agentId back. On-chain identity created.
registerAgent("trading-bot",
"trader.agentcover.eth")
→ agentId: 3
Eth Sepolia
Set ENSIP-25 Text Record
On the ENS resolver, set a text record keyed by the ERC-7930 encoded address of the registry + agentId.
setText(node,
"agent-registration[0x0001...14a54][3]",
"1")
Cross-Chain
Verify Ownership
Protocol reads both chains: text record exists on ENS, agent is active on Base, ENS names match, owner matches.
verifyEnsip25(
  "trader.agentcover.eth",
  registryAddr, 3, owner
) → { verified: true }

SDK Quick Start
// register + verify in one script
import { AgentCover } from '@agentcover/sdk'

const ac = new AgentCover({
  baseSepolia: 'https://sepolia.base.org',
  ethSepolia: 'https://sepolia.eth.org',
})

// 1. Claim ENS subname
await ac.claimENS('trader')
// → trader.agentcover.eth

// 2. Register on Base
const { agentId } = await ac.register({
  name: 'trading-bot-v2',
  ens: 'trader.agentcover.eth',
})

// 3. Cross-chain verify
await ac.verify(agentId)
// sets ENSIP-25 text record

// 4. Get coverage
await ac.insure({
  agentId,
  coverage: 50_000,
  risks: ['contract', 'oracle'],
})
// → premium: 12.36 USDC/mo
